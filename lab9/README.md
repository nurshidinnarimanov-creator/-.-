### Цель работы

---

### Запуск приложения

Открыть файл index.html в браузере.

---

### Краткий теоретический обзор

Fetch API — современный интерфейс браузера для выполнения HTTP-запросов.
Основные особенности:

Возвращает Promise, что позволяет удобно использовать async/await.
Поддерживает большинство типов запросов: GET, POST, PUT, DELETE.
Предоставляет метод response.json() для обработки JSON-данных.
Позволяет перехватывать сетевые ошибки и обрабатывать некорректные ответы сервера.
